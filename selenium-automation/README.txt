# Selenium Automation: Demo Bookstore Screenshot Capture

## What this does
This Python script uses Selenium to:
- Open the demo bookstore website (https://demo.nopcommerce.com)
- Simulate user actions (register, login, search, cart)
- Capture screenshots at each step

## Requirements
- Python 3.x
- Google Chrome browser
- ChromeDriver (same version as Chrome)
- `pip install selenium`

## How to Run
1. Place `chromedriver.exe` in this folder or in your system PATH.
2. Run the script:
   ```bash
   python capture_screenshots.py
   ```
3. Check the `Screenshots/` folder for results.
